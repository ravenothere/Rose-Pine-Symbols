## Installation
Available on:
- **[Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=ravenothere.rose-pine-symbols)**

- **[Open VSX Registry](https://open-vsx.org/extension/ravenothere/rose-pine-symbols)**

1 ⁝ Open VS Code.

2 ⁝ Go to the Extensions view (Ctrl+Shift+X).

3 ⁝ Search for `Rosé Pine Symbols`.

4 ⁝ Click Install.

## Configuration

You can choose your folder style from the settings:

```json
"rose-pine-symbols.folderStyle": "rose-pine"
```

| value | description |
|---|---|
| `rose-pine` | Rosé Pine arrow folder icons — explorer arrows hidden |
| `symbols` | Symbols folder icons in Rosé Pine colors — explorer arrows visible |

A window reload is required after changing this setting.
